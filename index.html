Portfolio

Grime Lord

Grime Lord is a mobile game about color theory for iOS and Android. I created the game concept and mechanics, designed all the game elements in Illustrator, and programmed it in react-native.

Built With

React-Native, Redux, Redux-Saga, React-Native-SVG, Illustrator

Globaleyes

globaleyesgame.com

Globaleyes is a browser based educational platform for teaching geography, civics, and history in a fun and unique way. It leverages the Google Maps API to create global virtual scavenger hunts that teach about different cultures, sites, and historical events.

Built With

React, Redux, HTML, SASS, Google Maps API, Firebase
